Apache Isis™ is a platform to let you rapidly develop domain-driven apps in Java.

This example app demonstrates the Incode' Classification Module, configured for a very simple and purposefully minimal application. The application demonstrates the ability to classify any (arbitrary) domain object against a pre-defined set of categorizations available for that type of domain object. A CharacterizationCategory defines a particular classification (similar to an enum), eg "PrivilegeLevel" and applies to a specified domain type (eg Customer). A CharacterizationValue is a particular value within that classification (similar to an enum instance), eg "PrivilegeValue#GOLD", and applies to a specific instance of the domain type (eg "John Doe: Customer").

The module depends on the poly module (polymorphic association pattern).

For more details, see the github repo.
